# Mini Banking System

Prosty system bankowy napisany w PHP.

## Funkcje

- Konto osobiste (bez debetu)
- Konto firmowe (z limitem debetu)
- Wpłaty i wypłaty
- Przelewy między kontami
- Obsługa błędu braku środków (wyjątek)

---

## Wymagania

- PHP 8.1+
- Composer (opcjonalnie, ale zalecany)

---

git clone https://github.com/piotrmarczewski/task-1-banking-system.git
